Available on all platforms

FlxZoomCamera
FlxZoomCamera

ZoomCamera: A FlxCamera that centers its zoom on the target that it follows Flixel version: 2.5+

The camera class is used to display the game's visuals on the screen. By default one camera is created automatically, that is the same size as the screen. You can add more cameras or even replace the main camera using utilities in FlxG.

var deadzone:FlxRect = null

You can assign a "dead zone" to the camera in order to better control its movement. The camera will always keep the focus object inside the dead zone, unless it is bumping up against the bounds rectangle's edges. The deadzone's coordinates are measured from the camera's upper left corner in game pixels. For rapid prototyping, you can use the preset deadzones (e.g. STYLE_PLATFORMER) with follow().

Handy function for "killing" game objects. Use reset() to revive them. Default behavior is to flag them as nonexistent AND dead.
