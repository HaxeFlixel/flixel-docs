loadGraphic(Graphic:FlxGraphicAsset, Animated:Bool = false, Width:Int = 0, Height:Int = 0, Unique:Bool = false, ?Key:String):FlxSprite

Load an image from an embedded graphic file.

Parameters:

Graphic

The image you want to use.

Animated

Whether the Graphic parameter is a single sprite or a row of sprites.

Width

Optional, specify the width of your sprite (helps FlxSprite figure out what to do with non-square sprites or sprite sheets).

Height

Optional, specify the height of your sprite (helps FlxSprite figure out what to do with non-square sprites or sprite sheets).

Unique

Optional, whether the graphic should be a unique instance in the graphics cache. Default is false.

Key

Optional, set this parameter if you're loading BitmapData.

Returns:

This FlxSprite instance (nice for chaining stuff together, if you're into that).
